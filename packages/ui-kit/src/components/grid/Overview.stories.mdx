## **Grid**

To create the layout for page in proper structure for easy maintenance.

#### **Component Properties**: 
We are using flex based Row-Columns.
- If you want full height row you can use  h-full  class
- If you want vertical columns you can use  flex-col  class in parent row
- If you want horizontal columns (like rows) you can use  flex-row  class in parent row
- For columns you can pass  flex as a prop  to set width or height of column, if parent have flex-col it will set width for column and if parent have flex-row it will set height of column
- You can pass overflow - ‘auto, hidden, visible’ as a prop in the column.
- By default scrollbar is hidden, if you want to show it you have to pass   visible-scrollbar  class (it will only work with overflow-auto)

#### **Component Related**:

- RootContainer
- Resizable

#### **Component Dependencies**:

- Use tailwind classes for flex to generate any layout you want.

#### **Best Practices**:

- Always nest ```<Column></Column>``` in ```<Row></Row>```

Do: 

```
<Row>
	<Column>
	Loream ipsum unde omnis iste natus error sit voluptatem accusantium.
	</Column>
	<Column>
		<Row>
			<Column>
				Loream ipsum unde omnis iste natus error sit.
			</Column>
      <Column>
				Loream ipsum unde omnis iste natus error sit.
			</Column>
		</Row>
	</Column>
</Row>
```
Avoid:

```
<Row>
  <Row>
    Do not nest unnecessary Rows
  </Row>
</Row>
```

```
<Row>
	<Column>
		<Column>
			Do not nest unnecessary Rows
		</Column>
	<Column>
</Row>
```

```
<Row>
	Do not place any component directly in Row, use Column inside a Row.
</Row>
```
