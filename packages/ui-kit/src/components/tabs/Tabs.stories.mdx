## **Tabs**

To create view for single tab.

- Is used in multiple locations [ within tab header, body tab headers, and so ]

#### **Component Usage**:

- Tabs can be updated with component on preComp(start) and postComp(end)
- Horizontal scroll will be enabled by default if tabs rendered exceed the available width

#### **Best Practices**:

- For preComp, postComp, suffix - create functional component
- Example: ```() => <div>Icon</div>```

Do:

```
 <Tabs list={[]}
       activeTab={activeTab}
        onSelect={() => {}}
        closeTabIconMeta={{
            show: true,
            onClick: (i, id) => removePlayground(id),
          }}
      suffixComp={() => {
            return (
              <Button
                text="suffix btn"
                secondary
                iconLeft
                sm
                transparent={true}
                ghost={true}
              />
            );
          }}
      preComp={() => {
            return (
              <Button
                text="preFix btn"
                secondary
                iconLeft
                sm
                transparent={true}
                ghost={true}
              />
            );
          }}
      postComp={() => {
            return (
              <Button
                text="postFix btn"
                secondary
                iconLeft
                sm
                transparent={true}
                ghost={true}
              />
            );
          }}
        />
```
