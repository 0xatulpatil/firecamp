## **Copy Button**

- Copy button can be used with following sub components
  - text
  - custom component

#### **Component Usage**

- We can add text before the copy button icon
- We can show/hide text preview `copied` based on click event
- On adding custom component, the copy functionality gets disabled.

Copy Button Preview

| Version | Preview |
| ---- | ---------------------------------- |
| default | <Button showText={false}/> |
| with text | <Button text="Copy Button" showText={true}/> |
| custom component | <Button>[<p>Custom Component</p>]</Button> |
