## **Tab**

View for single tab in list.

#### **Component Usage**:

- Single Tab can contain elements like

  - Close Icon
  - Count
  - With Icon

- Used within TabList component.

| Version   | Preview                                                                        |
| --------- | ------------------------------------------------------------------------------ |
| active    | <Tab name='Sample Tab' isActive/>                                              |
| count     | <Tab name='Sample Tab' count="5"/>                                             |
| with icon | <Tab name='Sample Tab' count="5" preComp={() => <Bell size={24}/>} /> |
