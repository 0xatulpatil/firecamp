## **Tabs**

To create view for list with single/multiple tab.

- Is used in multiple locations [ within tab header, body tab headers, and so. ]

#### **Component Usage**:

- Tabs can be updated with component on preComp(start) and postComp(end)
- Horizontal scroll will be enabled by default if tabs rendered exceed the available width

#### **Best Practices**:

- For preComp, postComp, suffix - create functional component
- Example: `() => <div>Icon</div>`

Do:

```
 <Tabs list={[]}
       activeTab={activeTab}
        onSelect={() => {}}
        closeTabIconMeta={{
            show: true,
            onClick: (i, id) => removePlayground(id),
          }}
      suffixComp={() => <Button text="suffix btn" sm />}
      preComp={() => <Button text="preFix btn" sm/>}
      postComp={() => <Button text="postFix btn" sm/>} />
```
