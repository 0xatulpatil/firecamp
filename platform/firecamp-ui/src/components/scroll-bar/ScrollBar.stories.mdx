## **ScrollBar**

To display the custom scrollbar instead of default.

#### **Component Usage**:

- Wrap the component to display the scrollbar

- Provide the width & height to the classname to display scrollbar on reaching specific dimension

| Prop      | Preview                                                  |
| --------- | -------------------------------------------------------- |
| className | add classname to ScrollArea                              |
| width     | Add width for horizontal scrolling                       |
| height    | Add height for vertical scrolling                        |
| noWrap    | To prevent content wrap for displaying Horizontal scroll |
