## **ScrollBar**

To display the custom scrollbar instead of default.

#### **Component Usage**:

- Wrap the component to display the scrollbar

- Provide the width & height to the classname to display scrollbar on reaching specific dimension Or provide fullWidth/fullHeight 

| Prop        | Preview                                                                                 |
| ----------- | --------------------------------------------------------------------------------------- |
| children    | content for ScrollArea                                                                  |
| className   | add classname to ScrollArea (add h-* or w-* incase fullWidth/fullHeight is not defined) |
| noWrap      | To prevent content wrap to next line for displaying Horizontal scroll                   |
| withCorner  | To keep the corner space where both scollbar intersect                                  |
| transparent | To keep the srollbar track as transparent                                               |
| fullWidth   | Add 100% width for horizontal scrolling                                                 |
| fullHeight  | Add 100% height for vertical scrolling                                                  |
| rounded     | To keep the scrollbar radius as rounded                                                 |
