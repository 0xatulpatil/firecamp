## **ScrollArea**

The ScrollArea component allows you to display a custom scrollbar instead of the default browser scrollbar. It extends the functionality of the Mantine ScrollArea component.

#### **Component Usage**:

- Wrap your content with the `ScrollArea` component to enable the custom scrollbar.
- Provide a maximum width/height to the parent component to trigger the appearance of the scrollbar when content overflows.
- Alternatively, you can provide the `w` (width) and `h` (height) props directly to the `ScrollArea` component.
- If you wish to display only the vertical scrollbar, ensure that the child contents do not overflow along the horizontal axis.
