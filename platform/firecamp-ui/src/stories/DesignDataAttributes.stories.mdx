## **CSS Classes:**

- We have defined data-\* attributes to save element value , which are listed below:

| Element    | Attribute          | Description                                        |
| ---------- | ------------------ | -------------------------------------------------- |
| div        | data-tip           | to value for tooltip text                          |
|            | data-for           |                                                    |
|            | data-testid        | to find the element while running test cases       |
| Button     | data-testid        | to find the element while running test cases       |
| a          | data-tip           | to value for tooltip text                          |
|            | data-multiline     |                                                    |
|            | data-place         |                                                    |
| svg        | data-tip           | `react-icons`                                      |
|            | data-for           | `react-icons`                                      |
| span       | data-testid        | to find the element while running test cases       |
|            | data-testid        | `@monaco-editor/react`                             |
| table - th | data-allow_resize  | to test the column for resizing                    |
|            | data-initial_width | to test the column width for maintaining min width |
