## **CSS Design Variable:**

- We have two themes defined
  - theme-dark
  - theme-light
- Default theme is `theme-dark`

- In each theme, we are having 2 primary colors defined
  - primary-orange
  - primary-green
- Default primary color is `primary-orange`

- To apply theme, theme & primary color is required to be defined `<body>` tag classname

- Same color variables are required to be defined for both themes
