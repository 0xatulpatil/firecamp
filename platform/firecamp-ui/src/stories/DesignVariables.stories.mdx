## **CSS Design Variable:**

- We have two themes defined
  - theme-dark
  - theme-light
- Default theme is `theme-dark`

- In each theme, we are having 2 primary colors defined
  - primary-orange
  - primary-green
- Default primary color is `primary-orange`

- To apply theme, theme & primary color is required to be defined `<body>` tag classname

- Same color variables are required to be defined for both themes

- Example of body tag classname : `<body className="theme-dark primary-orange"/>`

### Initial Values

- For `<body>`, initial values defined below:

| Property                           | value                                       |
| ---------------------------------- | ------------------------------------------- |
| Body background color & text color | @apply bg-appBackground text-appForeground; |
| height                             | 100vh                                       |
| width                              | 100vw                                       |
| margin                             | 0px                                         |
| padding                            | 0px                                         |
| -moz-user-select                   | none                                        |
| -webkit-user-select                | none                                        |
| -ms-user-select                    | none                                        |
| user-select                        | none                                        |
| -o-user-select                     | none                                        |
| background                         | var(--app-background)                       |
| color                              | var(--app-foreground)                       |
| font-weight                        | 400                                         |
| font-family                        | var(--font-family)                          |
