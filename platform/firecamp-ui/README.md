# Run
Run Storybook:
```
pnpm storybook
```
Run Jest test cases :
```
pnpm test
```
Run the Storybook after running all available test cases:
```
pnpm start:storybook
```

By default, a server at [localhost:6006](http://localhost:6006/) is started.
